# Scraping-Wordcloud
네이버 지식인 검색어 스크래핑 후 워드클라우드 이미지(알리, 테무, 중국쇼핑 검색어) 생성 프로젝트 

-------
사용 라이브러리 및 버전
1. python     3.10.14
2. beautifulsoup4     4.12.3
3. matplotlib      3.8.4
4. numpy      1.26.4
5. pandas     2.2.2
6. PyQt5     5.15.10
7. requests      2.31.0
8. selenium      4.20.0
9. Qt designer     5.11.1
10. wordcloud      1.9.3
